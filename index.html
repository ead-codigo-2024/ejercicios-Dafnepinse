Diseño desde el código 2024

Portafolio con ejercicios de Dafne Pincheira, S2 2024

Ejercicios

En esta asignatura, aprendimos a usar HTML, CSS y JavaScript para crear sitios web. Empezamos viendo los conceptos básicos de HTML y luego avanzamos a temas como cómo estructurar una página, usar hojas de estilo, organizar el diseño con CSS y hacer que la página se vea bien en diferentes dispositivos. También trabajamos con JavaScript para añadir interactividad, formularios y aprendimos sobre accesibilidad para que las páginas sean fáciles de usar para todos. Al final, aplicamos todo lo aprendido en un proyecto final, la creación de una página web.
